使い方

オーナーの作業は素材を送るだけ

動画素材を送る

施術動画をLINEまたは専用フォームから送信するだけ。撮影ガイドラインに沿った素材であれば品質が向上します。

AIがコンテンツを生成

ブランドガイドラインに基づき、AIがキャプション・ハッシュタグ・投稿スケジュールを自動作成します。

承認または自動投稿

事前確認モードまたはお任せモードを選択。承認モードでは投稿前にLINEで内容を確認できます。

全チャンネルに同時配信

Instagram・YouTube・TikTok・Facebook・ThreadsにAPIを通じて自動投稿。結果はWeeklyレポートで確認できます。

対象サービス

本アプリは以下のSNSプラットフォームのAPIを利用します

🎵

TikTok Content Posting API

サロン顧客の許可を得た上で、TikTokアカウントへの動画投稿を自動化します。OAuth認証により安全に連携します。

📸

Meta Graph API

Instagram Reels・Facebook・ThreadsへのコンテンツをMeta公式APIで配信します。

▶️

YouTube Data API

YouTube ShortsへのアップロードをYouTube Data API v3で自動化します。
